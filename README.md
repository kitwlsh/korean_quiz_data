# KDailyUtil 퀴즈 데이터 저장소

KDailyUtil 앱에서 사용하는 퀴즈 데이터를 자동으로 생성하고 관리하는 저장소입니다.

## 🤖 AI 자동 업데이트 시스템

본 저장소는 GitHub Actions를 통해 매일 자동으로 새로운 퀴즈를 생성합니다.

- **실행 스크립트**: `update_quiz.py`
- **사용 모델**: `gemini-2.5-flash` (사용자 환경 최적화 모델)
- **업데이트 주기**: 매일 자정 (UTC 00:00)
- **데이터 분류**:
  - `korean.json`: 우리말 겨루기 (맞춤법, 고유어, 사자성어)
  - `trend.json`: 트렌드 말하기 (신조어, 유행어)
  - `knowledge.json`: 상식 백과 (역사, 과학, 시사)
  - `travel.json`: 세계 여행 (국가수도, 국기, 명소)

## 🛠 수동 업데이트 방법

새로운 문제를 즉시 생성하고 싶다면 다음 단계를 따르세요:

1. GitHub 저장소의 **[Actions]** 탭으로 이동합니다.
2. **'매일 퀴즈 업데이트 로봇'** 워크플로우를 선택합니다.
3. **[Run workflow]** 버튼을 클릭하여 즉시 실행합니다.

## 📌 참고 사항
- 모델명 `gemini-2.5-flash`는 현재 사용자 환경에서 가장 안정적으로 작동하는 모델로 설정되어 있습니다. 모델 변경이 필요할 경우 `update_quiz.py` 파일을 수정하세요.
